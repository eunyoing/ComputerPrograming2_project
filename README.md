# 🎮 DODGE AND ESCAPE!

장애물을 피하고 탈출구에 도달하라!  
Java로 만든 2D 플랫폼 점프 게임

---

## 📌 프로젝트 개요
사용자가 이름을 입력하고 난이도를 선택해 게임을 즐길 수 있는 2D 플랫폼 점프 게임입니다.  
캐릭터를 조작해 장애물을 회피하며 탈출구에 도달하는 것이 목표이며,  
랭킹 시스템을 통해 자신의 성과를 기록하고 확인할 수 있습니다.

---

## 🛠 사용 기술
- 언어: Java
- 개발 도구: IntelliJ IDEA
- UI 라이브러리: Java Swing
- 네트워크: 클라이언트-서버 소켓 통신

---

## 🎯 주요 기능
- 사용자 이름 입력 및 난이도 선택 (Easy / Normal / Hard)
- 난이도별 맵 및 장애물 구성
- 게임 클리어 / 게임 오버 화면
- 난이도별 랭킹 시스템
- 튜토리얼 모드
- 클라이언트-서버 기반 랭킹 통신

---

## 👥 역할 분담

| 이름 | 담당 역할 |
|------|----------|
| 황은영 | 메인 스크린, 디자인, 클리어/오버 화면, 랭크 기본 구조 |
| 박준형 | Easy 맵, 튜토리얼, 랭킹 시스템, 난이도 조절 |
| 허은철 | Normal/Hard 맵, 서버 구현, 코드 정리 |

---

## 🗂 데이터 저장 구조
게임 종료 시 사용자 이름, 난이도, 점수를 저장하고  
난이도별로 정렬하여 랭킹을 업데이트합니다.

---

## 🌐 네트워크 구조
클라이언트 → 난이도 전송  
서버 → 랭킹 조회 후 반환
